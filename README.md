# dataTraffic
データ残量をLINEに通知するスクリプト
## Setup
```git clone https://github.com/0x0u/dataTraffic```  
```cd dataTraffic```  
```pip3 install -r requiremenets.txt```  
dataTraffic.pyをエディターで開きtelnum、password、access_tokenにそれぞれ自分のものに置き換える。  
```python3 dataTraffic.py```
